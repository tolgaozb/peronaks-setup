# PeronAKS — Guncelleme / Dagitim

Bu depo **yalnizca calistirilabilir dosyayi (PeronV4.exe)** dagitmak icindir.
Kaynak kod burada **degildir** (ozel/private deposunda tutulur).

## En son surumu indir

Her zaman en guncel exe:

    https://github.com/tolgaozb/peronaks-setup/releases/latest/download/PeronV4.exe

Tum surumler: **Releases** sekmesine bakin.

## Kurulum / Guncelleme

1. Yukaridaki linkten `PeronV4.exe` dosyasini indirin.
2. Programi kapatin.
3. Mevcut `PeronV4.exe` yerine yenisini kopyalayin.
4. Programi tekrar acin.

> Not: Veritabani (`PerCore.mdb`) ve ayar dosyalari (`vdzSabit.ini`) degismez; sadece exe guncellenir.
