# PeronAKS — Guncelleme / Dagitim

Bu depo **yalnizca calistirilabilir dosyalari** dagitmak icindir.
Kaynak kod burada **degildir** (ozel deposunda tutulur).

## En son surumu indir

**Tam paket** (exe + bagimli dosyalar) — onerilen:

    https://github.com/tolgaozb/peronaks-setup/releases/latest/download/PeronAKS.zip

**Sadece uygulama** (hizli guncelleme):

    https://github.com/tolgaozb/peronaks-setup/releases/latest/download/PeronV4.exe

Tum surumler ve surum notlari: **Releases** sekmesi.

## Paket icerigi (PeronAKS.zip)

- PeronV4.exe — ana uygulama
- AksReader Library 3.0.dll + .tlb — okuyucu kutuphanesi
- ConnBul.exe — baglanti bulma araci

## Kurulum / Guncelleme

1. Programi kapatin.
2. Paketteki dosyalari mevcut kurulum klasorune kopyalayin (uzerine yazin).
3. Programi tekrar acin.

> Onemli: Veritabani (PerCore.mdb) ve ayar dosyasi (vdzSabit.ini) pakette
> yoktur ve degismez -- mevcut verileriniz korunur.
